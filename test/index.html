Installation

Simply include javascript-key-codes.js in your page, and then pass e.which as an argument to the get_key() function.

The get_key() function will return a human-readable key. In the following example we can say "if key == 'enter'" and then do something if the user pressed the enter key.

$(document).ready(function(){
	$(document).on('keyup', function(e){
		var key = get_key(e.which);
		
		$('#result').html(key);
		
		if(key == 'enter'){
			alert('You pressed enter');
		}

		return false;
	});
});

All key responses are in lower case. Pressing the backspace key will return 'backspace'. The keys and values are stored in an array in the javascript file, so if you can't find out what something is referenced as, you can check there.
